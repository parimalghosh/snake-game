# snake-game

Full feature breakdown:

Canvas-rendered smooth gameplay with rounded snake segments and gradient body fade
Snake eyes that follow the direction of movement
3 game modes — Walls Kill (classic), Wrap Around (pass through walls), Speed Rush (fast wrap mode)
3 power-ups that spawn randomly as diamonds:

⚡ Speed Boost — snake moves faster for bonus challenge
❄ Slow-Mo — slows down for easier control
✦ 2X Points — doubles your score per food


Power-up timer bar at the top showing remaining duration
Leveling system — speed increases every 50 points
Particle explosions on eating food, grabbing power-ups, and on death
Neon glow trail that follows behind the snake
Pulsating food with glow effects and floating power-up diamonds
Sound effects — synthesized tones for eating, power-ups, leveling, and death
High score persistence saved in localStorage
Pause/Resume with Space key or pause button
Mobile swipe controls with tap-to-resume
WASD + Arrow key support
Score pop animation showing +10 or +20 on each food pickup
Game Over screen with final stats and new high score detection
